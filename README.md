# ربات فروشگاهی تلگرام (aiogram 3.x)

این پروژه یک ربات فروشگاهی تلگرام با معماری ماژولار است که با **Python 3.12+** و **aiogram 3.x** پیاده‌سازی شده و از **SQLite (aiosqlite)** برای ذخیره اطلاعات استفاده می‌کند.

## قابلیت‌های فعلی

- ثبت خودکار کاربر در `/start`
- سیستم زیرمجموعه‌گیری (Referral) با پاداش خودکار
- کیف پول کاربر و شارژ موجودی
- خرید سرویس از دسته‌بندی‌ها با کسر موجودی
- ثبت تاریخچه تراکنش‌ها و خریدهای اخیر
- پنل ادمین برای:
  - افزودن کانفیگ جدید
  - مدیریت قیمت دسته‌بندی‌ها
  - مشاهده آمار موجودی، فروش و کاربران
  - تایید/رد رسیدهای شارژ
  - تنظیم شماره کارت مدیر از داخل ربات
  - ارسال پیام همگانی (Broadcast)

## پیش‌نیازها

- Python 3.12 یا بالاتر
- توکن ربات از BotFather
- SQLite (به صورت پیش‌فرض با پایتون در دسترس است)

## نصب و اجرا

### 1) ساخت محیط مجازی و نصب وابستگی‌ها

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install aiogram aiosqlite pydantic-settings
```

### 2) تنظیم فایل `.env`

یک فایل `.env` در ریشه پروژه بسازید:

```env
BOT_TOKEN=PASTE_YOUR_BOT_TOKEN_HERE
ADMIN_ID=123456789
SUPPORT_ID=@support
ADMIN_CARD_NUMBER=6037-xxxx-xxxx-xxxx
```

> `ADMIN_ID` باید آیدی عددی تلگرام ادمین اصلی باشد.

### 3) اجرای ربات

```bash
python3 bot.py
```

## دیتابیس

فایل دیتابیس پیش‌فرض: `database.db`

جداول اصلی که به صورت خودکار ساخته می‌شوند:

- `users` (کاربران، موجودی، معرف)
- `configs` (کانفیگ‌های قابل فروش)
- `categories` (دسته‌بندی و قیمت)
- `transactions` (تراکنش‌های شارژ/خرید)
- `settings` (تنظیمات کلیدی مانند شماره کارت ادمین)

> در شروع برنامه، اگر مقدار `admin_card_number` در جدول `settings` موجود نباشد، از مقدار `.env` مقداردهی اولیه می‌شود.

## نکات مهم

- فایل `.env` را در گیت push نکنید.
- در اولین اجرا، تمام جداول لازم به شکل خودکار ایجاد یا مهاجرت داده می‌شوند.
- پیام‌ها و کیبوردها فارسی هستند و از `parse_mode` های HTML/Markdown استفاده می‌کنند.

## توسعه‌های پیشنهادی

- افزودن `requirements.txt` برای نصب یک‌مرحله‌ای
- اضافه کردن Dockerfile برای استقرار راحت‌تر
- اضافه کردن لاگ‌گیری ساختاریافته و مانیتورینگ
- افزودن تست برای لایه دیتابیس و هندلرهای کلیدی
